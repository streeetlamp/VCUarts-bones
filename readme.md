## Getting Started
```bash
git clone git@github.com:streeetlamp/bare-wp.git
```

```bash
npm install
```

## Grunt Tasks
*Watching + Live Reloading of JS, CSS, PHP*
```bash
grunt build
```

```bash
grunt dev
```

*Build minified CSS + JS*
```bash
grunt build
```

*WP SNIFFIN*
```bash
grunt sniff
```
